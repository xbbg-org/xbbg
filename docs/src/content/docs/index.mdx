## Quick Example

```python
import xbbg

# Reference data
df = xbbg.bdp('AAPL US Equity', ['PX_LAST', 'VOLUME'])

# Historical data  
df = xbbg.bdh('AAPL US Equity', 'PX_LAST', start_date='2024-01-01')

# Use Polars instead of pandas
xbbg.set_backend('polars')
df = xbbg.bdp('AAPL US Equity', 'PX_LAST')
```
